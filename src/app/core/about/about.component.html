Description
The purchase request system allows company employees to submit requests to purchase items from specific vendors with approval from management as determined through business rules.
User Features
  • All users can view data from all tables. All users can create requests and edit and delete their own requests.
  • Base level users can only view their own requests.
  • Reviewers can view all requests and have access to the review tab which allows them to reject or approve all requests except their own requests.
  • Admins have access to create, edit and delete data from any table.
